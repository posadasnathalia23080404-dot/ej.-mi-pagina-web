LO QUE SE APRENDERÁ EN LA CLASE DE CONSTRUYE APLICACIONES WEB

Tema 1: Conocer las etiquetas básicas de HTML y su estructura.

Tema 2: Aprender a utilizar CSS para darle estilo y personalidad a una página.

Tema 3: Integrar imágenes, videos y audio dentro de un sitio web.

Tema 4: Crear formularios y capturar información del usuario.

Tema 5: Manejar diferentes tipos de diseño como tablas, listas y divisiones.

Tema 6: Entender cómo funciona el modelo cliente-servidor y la navegación web.

Tema 7: Subir un sitio web a internet y mantenerlo actualizado.

Tema 8: Usar buenas prácticas de programación para páginas limpias y ordenadas.

Tema 9: Introducción a JavaScript para añadir interactividad.

Tema 10: Construir páginas completas y funcionales desde cero.

Opinión

En mi opinión, esta materia es muy útil porque enseña habilidades reales que se aplican fuera de la escuela. Aprender a crear aplicaciones web abre muchas oportunidades, permite desarrollar creatividad y también aprender lógica. Además, es una de las áreas más importantes hoy en día porque casi todo está en internet. Mientras avances con paciencia y practiques, podrás crear páginas cada vez más profesionales.
