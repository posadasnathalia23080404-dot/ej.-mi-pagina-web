Ejemplos de las benzodiacepinas que existen

EjemplosPara que sirvenImagenes
Alprazolam Conocido por nombres comerciales como Xanax®, se utiliza para la ansiedad.
Clonazepam Tambien llamada Klonopin o Rivotril, es comun para tratar el pánico y las convulsiones.
Diazepam Conocido como Valium®, es una benzodiazepina de acción prolongada.
Lorazepam Conocido como Ativan® u Orfidal®, se utiliza para la ansiedad y las convulsiones.
Midazolam También llamado Versed®, se usa para sedación y anestesia.
Temazepam Conocido como Restoril®, se usa para el insomnio.
Triazolam Conocido como Halcion®, se utiliza para tratar el insomnio a corto plazo.
Bromazepam Se utiliza para tratar la ansiedad, la tensión, el nerviosismo y, en dosis más altas, como sedante y relajante muscular.
